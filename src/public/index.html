Create an Ephemeral Channel

We invite you to participate in consentful, transparent communications within your community.

When you create a new channel, you are able to choose from three different meeting modes – Facilitated Mode, Feedback Mode, and Egalitarian Mode, each bundled with different features.

Name

Set the name of your channel around a discussion topic (max 25 characters).

Meeting Modes

Facilitated Mode

The channel creator can appoint 0~3 facilitator(s) in this mode. If selected, the channel will have access to the Amplify Module (✦) and the Consentful Gesture Module (⇆).

The channel creator can appoint 1~3 facilitator(s) in this mode. If selected, the channel will have access to the Amplify Module (✦) and the Voting Module (✓).

Amplify Module (✦)

If a facilitator is appointed, they will be able to use the amplify module (✦) to outline agendas, ask questions and guide conversations within the channel.

Voting Module (✓)

Participants will be able to use the voting module (✓) to turn any message into a poll under the text panel.

Consentful Gesture Module (⇆)

Participants will be able to provide secondary feedback under each message such as “agree” (⇈), “hesitant” (⇄), “disagree” (⇊), or “block” (ᳵ) to signal their support or concerns.
